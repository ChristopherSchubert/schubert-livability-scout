# Retired prototype mockups

These are **old, static design prototypes** — superseded HTML mockups moved out
of the served `public/` web root on 2026-06-22 (issue #75) so they are no longer
reachable on the production domain. They are kept here only as design history.

**Do not wire these to anything.** If you want one back as a live surface, move
it deliberately and document why.

## What stayed live (intentionally NOT retired)

Two prototypes are real, API-wired surfaces and remain under `public/`:

- `public/city-detail-redesign.html` — design surface backed by `/api/mockup-data`.
- `public/mockups/trip-walkthrough.html` — the canonical trip-planning deck and
  Janice's active feedback surface, backed by `/api/walkthrough-feedback`.
